Basic example
=============

A simple example.

Load the package


.. code-block:: default

    from toy_pkg.calculus import multiply, divide









Run some calculation

Use the useless functions defined in
toy_pkg to run some calculations
and check the results...


.. code-block:: default

    result = multiply(1, 1)
    print(result)

    result = divide(1, 3)
    print(result)
    print(multiply(-2, 6))
